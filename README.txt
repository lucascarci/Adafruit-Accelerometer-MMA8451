DEMI PEPPAS & LUCA SCARCI - ADAFRUIT ACCELEROMETER USAGE

Comments in the code describe the functionality of each line. Code works with Fritzing Diagram provided in the repository The code is a modified script from adafruit.com - https://learn.adafruit.com/adafruit-mma8451-accelerometer-breakout/overview


Libraries are required to run the accelerometer:

MMA8451 LIBRARY: https://github.com/adafruit/Adafruit_MMA8451_Library
ADAFRUIT SENSOR LIBRARY: https://github.com/adafruit/Adafruit_Sensor



BlUE LEDs: Levels of Acceleration
WHITE LEDs: Position on axis (GYROSCOPE)


5V input into Accelerometer used (3V Does not work)
Total of 9 LEDs (4 White, 5 Blue) connected to Digital Output Pins

See documentation video here: https://vimeo.com/261927682
